How It Works

仕組みはシンプルです。

毎日自動巡回

国（e-Gov）と東京の自治体サイトの意見募集・縦覧ページを、プログラムが毎日巡回します。取得先はすべて役所の公式一次情報です。

締切を抽出

案件ごとに募集締切を読み取り、締切が近い順のボードとカレンダー配信（ICS）に変換します。

あなたに届く

このページを見るか、カレンダー購読をしておくだけ。「知らないうちに決まっていた」がなくなります。
